# **App Name**: SpendWise

## Core Features:

- User Authentication: Secure user authentication using Firebase Authentication with email and password for signup, login, and logout functionalities.
- Expense Tracking: Allow users to add new expenses with details like title, amount, category, date, and timestamp.
- Expense Management: Enable users to view, edit, and delete their expenses stored in Firestore.
- Dashboard Overview: Display a dashboard summarizing total expenses, monthly spending, and recent transactions.
- Categorization: AI Tool which infers the category for the expense using generative AI when a category is not provided, and then the expense can be stored in Firestore
- Financial Summarization: Generate human-readable summaries of financial habits and expense tracking.
- Interactive Charts: Visualize expense data using charts (e.g., bar charts, pie charts) powered by Chart.js, showing category-wise breakdown.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to represent trust and stability in managing finances.
- Background color: Light Grey (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and calls to action.
- Body and headline font: 'PT Sans', a modern humanist sans-serif providing good readability.
- Use flat, minimalist icons to represent expense categories, enhancing visual clarity.
- Employ a responsive, grid-based layout for a consistent experience across devices.
- Incorporate subtle animations for transitions and interactions to provide a smooth user experience.
